# 🍎 Fruit Information UI/UX Design

A modern and responsive **Fruit Information UI/UX Design** created in **Figma**. This project showcases both **desktop and mobile interfaces** with vibrant colors, clean layouts, and an engaging user experience for exploring different fruits and their nutritional information.

---

## ✨ Project Overview

This project is designed to provide users with an attractive way to learn about various fruits through visually appealing interfaces. Each fruit has its own dedicated page with a unique color theme, high-quality illustrations, and informative content.

The design is fully responsive with separate layouts for **Desktop** and **Mobile** devices.
